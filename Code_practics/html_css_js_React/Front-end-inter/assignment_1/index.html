Introduction to HTML

HTML (Hypertext Markup Language) is the standard markup language used to create web pages.

Difference between HTML5 and HTML

HTML5 is the latest version of HTML, introducing new features and improved semantics.

Semantic Tags

HTML5 introduced semantic tags like <header>, <nav>, and <footer> for better page structure. They give meaning to both the browser and the developer.

Meta Tags

A meta tag in HTML is a special element that provides metadata about a web page. It's typically placed within the head section of an HTML document and does not appear as visible content on the webpage. Block-Scoped

Block-Scoped Elements: Create separate, full-width blocks and start on a new line. Examples include <div>, <p>, and <h1>.

Inline-Scoped Elements:

Inline-Level Elements: Do not create new blocks, stay in the flow of the surrounding text, and appear on the same line as neighboring elements. Examples include <a>, <strong>, and <span>.

Introduction to CSS3

CSS3 (Cascading Style Sheets) is the latest version of CSS used for styling web pages.

Difference between CSS and CSS3

CSS3 includes new features like animations and rounded corners, enhancing styling capabilities.

Types of CSS

There are three types of CSS: inline, internal, and external.

Inline CSS

Inline CSS is used to apply a unique style to a single element.

Internal CSS

Internal CSS is used to define styles for multiple elements within a single HTML document.

External CSS

External CSS is used to separate styles from the HTML document, making it easy to apply consistent styles across multiple web pages.

CSS Height

The CSS height property is used to control the height of elements, such as divs, images, or text containers.

CSS Width

The CSS width property is used to determine the width of elements, allowing you to adjust their horizontal size.

CSS Padding

The CSS padding properties are used to create space within elements, affecting the area between the content and the element's border.

CSS Margins

The CSS margin properties are used to create space around elements, affecting the area outside of any defined borders.
