Complete Installation of the Tailwind css by using NPM(node package manager) #### Tailwind CSS installation _0_ `npm init -y` _1_ `npm install -D tailwindcss postcss autoprefixer` _2_ `npx tailwindcss init -p` Paste this in tailwind.config.js `*` _4_ Create a Folder by the name of "Tailwind" and in it , create a file "tailwind.css" and paste the tailwind code. _5_ `npm install vite`; scripts `dev` : "vite" at last run `npm run dev`to start the server.
